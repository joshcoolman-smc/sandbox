# Color Spec

**Date:** January 28, 2026
**Type:** Brand / Design System

## Overview
A comprehensive color palette and typography specification page showcasing a green-focused brand system. Features large type specimens and detailed color swatches with technical values.

## Key Features
- Three-panel typography showcase (Regular, Medium, Semi Bold)
- Large specimen letters (Aa, Ff, Xx)
- Full alphabet and character set display
- Primary color palette (6 colors)
- Secondary color palette (7 colors)
- Detailed color values (HEX, RGB, CMYK)
- Gradient backgrounds on type specimens

## Design Principles
- Black background with green accent color system
- Monospace font for technical values
- Card-based color presentation
- Clean typographic hierarchy
- Professional brand specification format

## How to Run
Open `index.html` in a browser or use:
```bash
npm run dev
```

## Dependencies
- Google Fonts: Outfit (400, 500, 600)
