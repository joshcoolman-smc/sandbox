# Day at a Glance

**Date:** January 28, 2026
**Type:** Timeline / UI Component

## Overview
A timeline-based daily schedule interface with colorful event blocks, checkboxes, and time markers. Designed to showcase a clean, modern approach to calendar visualization.

## Key Features
- Vertical timeline with hour markers
- Color-coded event cards (workout, shower, breakfast, email)
- Interactive checkboxes with completion states
- Current time indicator (08:18)
- Sun/moon markers for day boundaries
- Day summary statistics
- Create event button
- Back/Continue navigation buttons

## Design Principles
- Dark theme (#1a1a1a background)
- Vibrant accent colors (blue, orange, purple, pink)
- Card-based event design
- Clear visual hierarchy
- Rounded corners and modern spacing

## How to Run
Open `index.html` in a browser or use:
```bash
npm run dev
```

## Dependencies
- Google Fonts: Inter (400, 500, 600, 700)
