# Spec Sheet

**Date:** January 28, 2026
**Type:** Typography / Layout

## Overview
A bold typographic design experiment featuring a professional spec sheet layout with dark/light mode toggle. Emphasizes hierarchy through scale contrast and clean sectioning.

## Key Features
- Two-column grid layout with service categories
- Massive typography for "Elements & Principles"
- Dark mode (black/white) and light mode (white/black) toggle
- Horizontal dividers for visual rhythm
- Clean awards list presentation

## Design Principles
- Strong typographic hierarchy
- High contrast two-color palette
- Generous whitespace
- Professional and minimal aesthetic
- Emphasis on readability and structure

## How to Run
Open `index.html` in a browser or use:
```bash
npm run dev
```

## Dependencies
- Google Fonts: Work Sans (400, 700, 900)
