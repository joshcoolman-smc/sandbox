# Blend

**Date:** January 28, 2026
**Type:** Gradient / Color Exploration

## Overview
A gradient exploration experiment showcasing 12 distinct color blends organized into two style sets. Features a dramatic hero section with "Blend" wordmark and organized gradient galleries.

## Key Features
- Full-screen hero gradient with large title
- Two gradient style sets (6 gradients each)
- Style Set 01: Smooth linear gradients (Purple Sunrise, Twilight, Cosmic Burst, etc.)
- Style Set 02: Bold radial gradients (Nebula, Supernova, Eclipse, etc.)
- Named gradient swatches with labels
- 3-column responsive grid layout
- Black spacing sections between sets

## Design Principles
- Vibrant color combinations (purples, pinks, oranges, blues)
- Mix of linear and radial gradient techniques
- Consistent card proportions (1.4 aspect ratio)
- Labeled gradients for easy reference
- Black background for maximum contrast

## How to Run
Open `index.html` in a browser or use:
```bash
npm run dev
```

## Dependencies
- Google Fonts: Space Grotesk (700)

## Gradient Techniques
- Linear gradients at various angles (45°, 90°, 135°, 180°, 225°, 315°)
- Radial gradients with different focal points
- Multi-stop color transitions
