# NASA Layout

**Date:** January 29, 2026
**Type:** Typography and Layout

## Overview
NASA-inspired card layout featuring bold orange and black color scheme with diagonal stripe pattern. Recreates the aesthetic of modern space agency design with Off-White streetwear influence.

## Key Features
- Vibrant orange (#ff4500) primary card on dark background
- Large-scale typography with "NEW HORIZONS" title
- Bold NASA wordmark and agency description
- Quoted number "757" as focal point
- Diagonal black stripe pattern in lower half
- Small accent stripes above stripe section
- Rounded card corners with contained stripe overflow

## Design Principles
- High contrast between orange and black/dark gray
- Technical/industrial aesthetic
- Typography-driven layout
- Asymmetric information placement
- Layered stripe graphics for depth

## How to Run
Open `index.html` directly in a browser or use a local server.

## Dependencies
- Google Fonts: Space Grotesk (400, 500, 700 weights)
